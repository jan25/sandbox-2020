# Cobra todo cli app

```
mkdir cobra/todo

cobra init example --pkg-name="github.com/jan25/sandbox-2020/cobra/todo"
go mod init github.com/jan25/sandbox-2020/cobra/todo

```

## Features

```
todo list

todo add "next thing"

todo mark --done=0|1 "next thing"

todo purge --i-know-what-i-want
```
